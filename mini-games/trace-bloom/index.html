browser mini game · vanilla JavaScript · HTML Canvas

Trace Bloom

Draw a live signal trace through a corrupted network and stabilize leaking nodes before red noise pulses tear the route apart.

Move WASD / Arrows Trace Space / Left Mouse Pause P Restart R Mute M

How it works

Hold the trace button and move into cyan leak nodes to charge them. Stabilize all nodes to complete a wave. Red noise pulses damage your signal and can cut your active trace. The game gets denser and faster over time.
